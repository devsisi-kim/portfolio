# Videos 폴더

이 폴더에 프로젝트에서 사용할 비디오 파일들을 추가하세요.

## 필요한 비디오 파일들:

- `bodab-demo.mp4` - BODAB.AI 데모 비디오

## 지원 형식:
- `.mp4`
- `.webm`
- `.ogg`

## 사용 방법:
1. 비디오 파일을 이 폴더에 추가하세요.
2. 파일명은 정확히 위와 동일하게 해주세요. (대소문자 구분)
3. 또는 `constants.ts` 파일에서 마크다운 콘텐츠 내의 경로를 수정하여 원하는 파일명을 사용할 수 있습니다.

## 참고:
- 비디오는 마크다운 콘텐츠 내에서 이미지 문법으로 표시됩니다:
  ```
  ![비디오 설명](/videos/파일명.mp4)
  ```
